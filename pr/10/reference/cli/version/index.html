azd exec version

Display version information for the azd exec extension.

Synopsis

Shows the current version of the azd exec extension, build information, and compatibility details.

Flags

Flag Description
--help, -h Show help for this command

Examples

Show Version

Sample Output

Version Information

The version command displays:

  • Version number: Semantic version (e.g., 0.1.0)
  • Build date: When the extension was built (in development builds)
  • Commit hash: Git commit identifier (in development builds)
